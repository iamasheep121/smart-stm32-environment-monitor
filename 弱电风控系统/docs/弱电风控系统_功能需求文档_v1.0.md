# 弱电风控系统 - 功能需求文档

## 项目概述

弱电风控系统，根据弱电施工环境实现温度自动调节风扇转速，环境有害气体浓度超标预警，为施工环境提供一定安全保障。

## 用户故事

### 作为用户，我希望：

- 能够实时查看当前温度和湿度
- 风扇能够根据温度自动调节转速
- 可以手动控制风扇速度
- 能够设置温度阈值
- 系统状态清晰可见

## 功能列表

### 核心功能（必须实现）

- [x] 温度传感器数据采集和显示
- [x] PWM控制风扇转速（0-100%）
- [x] OLED显示系统状态
- [x] 自动/手动模式切换

### 进阶功能（争取实现）

- [x] 温度阈值设置和保存
- [x] 按键调节参数
- [ ] 系统自检功能
- [x] 串口调试接口

### 扩展功能（可选）

- [ ] 数据记录功能

- [x] 手机APP控制

- [ ] 多风扇协同控制

## 性能指标

- 温度测量精度：±1°C
- 控制响应时间：<2秒
- 系统稳定性：连续运行24小时无故障
- 功耗：待机<0.5W，全速<2W

## 约束条件

- 硬件平台：STM32F103C8T6
- 开发环境：Keil MDK + 标准库
- 开发仪器：万用表，示波器
- 开发周期：4周
- 成本预算：< 150元
